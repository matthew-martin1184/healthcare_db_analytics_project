Project Overview and Demonstration

This project develops an interactive dashboard in Streamlit for exploratory analysis of a healthcare provider network using data from a public-domain Kaggle dataset. It implements an end-to-end ETL pipeline, built in Jupyter Notebooks using Pandas, to transform raw CSV files into a structured relational schema and load the data into an Oracle Autonomous Data Warehouse. The data is queried using SQL and visualized through the dashboard to support insights into provider activity and patient encounters.

ETL Pipeline

This pipeline was developed in Jupyter Notebooks using Python, Pandas, Numpy, and Oracledb, to do the following:

  • Extract data from Kaggle API View Kaggle Dataset
  • Convert CSV files to Pandas DataFrames View Raw CSV Files
  • Clean and transform data in preparation for loading to Oracle Database View Cleaned CSV Files
  • Insert data into live Oracle Autonomous Data Warehouse

Future goals for the Healthcare Database Analytics Project:

Once the dashboard is fully developed with reporting functionality for exploratory analytics, I will implement a predictive modeling pipeline that integrates both regression and classification techniques. For quantitative features with sufficient cardinality, I plan to use multiple linear regression and multi-target linear regression to model provider capabilities. For categorical and lower-cardinality features, a logistic regression model will be trained to support binary and multiclass classification tasks.

  • Identify providers with the capacity to meet high-demand healthcare services
  • Classify patients based on the complexity of their medical needs
  • Match patients with providers who are best equipped to deliver appropriate care

The models will draw on a range of features, including provider specialization, availability, treatment history, and capacity, as well as patient characteristics. By scoring the alignment between patient needs and provider capabilities, the system will support data-driven care coordination, targeted resource deployment, and improved efficiency in matching supply with demand across the healthcare network.
